# PowerShell Automation Scripts

Collection of scripts to automate common Windows admin and troubleshooting tasks.

## 🚀 What’s Inside

- User creation & management scripts
- System health checks
- Service monitoring & restart scripts
- Network diagnostics helpers

## 🛠 How to Use

1. Download the script file(s)
2. Open PowerShell as Administrator
3. Run the script: `.\User-Creation.ps1`
4. Modify variables as needed

## 🎯 Why This Repo

This repository demonstrates my hands-on skills in Windows automation, troubleshooting, and system administration. Perfect for IT support and L2 engineers.
